## Release Versioning

We follow semantic versioning:

- **Major version (X.0.0):** Breaking changes, major framework redesigns
- **Minor version (0.X.0):** New features, significant additions (backward compatible)
- **Patch version (0.0.X):** Bug fixes, minor improvements, clarifications

**Current version:** 1.0.2 (Minor release with significant additions)
