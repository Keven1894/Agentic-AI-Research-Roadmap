## Style Guidelines

### Color Palette
- **Primary:** #2E86AB (Blue) - DIVA brand
- **Success:** #06A77D (Green) - Positive metrics
- **Warning:** #F18F01 (Orange) - Attention items
- **Neutral:** #666666 (Gray) - Secondary info
- **Background:** #F8F9FA (Light gray)

### Typography
- **Headings:** Inter Bold or Helvetica Bold
- **Body:** Inter Regular or Helvetica
- **Code/Data:** Fira Code or Consolas

### Chart Types
- **Comparison:** Grouped bar charts
- **Trends:** Line charts with markers
- **Composition:** Stacked bars or pie charts
- **Distribution:** Box plots or violin plots
